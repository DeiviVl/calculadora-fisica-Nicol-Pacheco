## 📋 Descripción del Proyecto

Este proyecto es una **Calculadora de Magnitudes Físicas** desarrollada como Actividad Evaluativa #3 para la materia **Construcción de Algoritmos** del programa **Ingeniería de Software**.

La aplicación web permite realizar **10 operaciones físicas fundamentales** con una interfaz moderna, interactiva y responsiva, construida con tecnologías web básicas: **HTML, CSS y JavaScript**.

### 🔢 Operaciones incluidas:

| # | Magnitud | Fórmula |
|---|----------|---------|
| 1 | Velocidad | `v = d / t` |
| 2 | Aceleración | `a = Δv / Δt` |
| 3 | Fuerza | `F = m · a` |
| 4 | Trabajo | `W = F · d · cos(θ)` |
| 5 | Energía Cinética | `K = ½ · m · v²` |
| 6 | Energía Potencial Gravitatoria | `U = m · g · h` |
| 7 | Densidad | `ρ = m / V` |
| 8 | Presión | `P = F / A` |
| 9 | Carga Eléctrica | `q = I · t` |
| 10 | Ley de Ohm | `V = I · R` |

### ✨ Características:

- 🎨 **Diseño visual atractivo** con paleta Blanco, Morado, Negro y Azul
- 📱 **Responsivo** — funciona en móvil, tablet y escritorio
- ⚠️ **Validaciones** — campos vacíos, divisiones por cero, valores negativos inválidos
- ⌨️ **Soporte teclado** — presiona `Enter` para calcular
- 🌟 **Partículas animadas** en el fondo
- 🗂️ **Navegación por pestañas** para cada calculadora

## 🚀 Instrucciones para Ejecutar

### Método 1 — Abrir directo en el navegador (sin servidor)

1. Descarga o clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/calculadora-fisica-nombre-apellido.git
   ```

2. Entra a la carpeta del proyecto:
   ```bash
   cd calculadora-fisica-nombre-apellido
   ```

3. Abre el archivo `index.html` en tu navegador favorito:
   - **Windows:** Doble clic en `index.html`
   - **macOS:** `open index.html`
   - **Linux:** `xdg-open index.html`

### Método 2 — Con servidor local (recomendado para desarrollo)

Si tienes **VS Code**, instala la extensión **Live Server**:
1. Abre la carpeta del proyecto en VS Code
2. Clic derecho en `index.html` → **"Open with Live Server"**

O con Python:
```bash
# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```
Luego abre: `http://localhost:8080`
